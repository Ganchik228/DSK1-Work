## Raspberry pi 4 B vlc

1. Установить образ с помощью `rufus` на microsd.

2. После установки образа, запустить raspberry pi и запустить `main.sh`:

    1. На рабочем столе правой кнопкой мыши нажать на файл `main.sh`
    2. Выбрать `OPEN`
    3. Выбрать `Execute in Terminal`

3. Выполнить указания установщика и дождаться перезагрузки системы.
4. Устройство готово к использованию.

пс. порт для подключения по ssh 33222.
